# 流程控制语句

### 条件语句

1. 只有一条语句的时候，可以不加大括号

   ```
   if(){
   }
   ```

   ```
   if(){
   }else{
   }
   ```

   ```
   if(){
   }else if{
   }else if{
   }else{
   }
   ```

2. switch case

   ```
   switch(n)
   {
   	case 1:
   		语句1;
   		break;
   	case 2:
   		语句2;
   		break;
   	default:
   		语句3;
   }
   ```

### 循环语句

1. while    先判断后循环，循环体有可能一次也不执行，循环体应包含使循环结束的语句

   ```
   while（判断语句）｛
   	循环体；
   ｝
   ```

2. do while    先运行后判断，至少运行一次，

   ```
   do
   {
      循环体;    
      循环增量;
   }while(循环条件)
   ```

3. for

   ```
   for(var i=0;i<10;i++){
   	console.log(i)
   }
   ```
